About Django Lab Project

This project demonstrates a fully configured Django web application designed for seamless serverless deployment on Vercel.

Using modern Bootstrap layouts, Font Awesome, and Python libraries, it serves as an excellent sandbox workspace for learning and creating administration dashboards or client portals.

Key Tech Stack Built-In:

  • Backend Framework: Django 6.0.6
  • CSS UI Styling: Start Bootstrap - SB Admin (Bootstrap 5)
  • Static Serving: WhiteNoise Integration
  • Serverless Hosting: Vercel Serverless Functions Runtime
